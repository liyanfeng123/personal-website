# 个人网站 / Personal Website

一个简洁现代的个人网站，包含**首页**和**关于我**两个页面。

## 📁 文件结构

```
personal-website/
├── index.html    # 首页
├── about.html    # 关于我
├── style.css     # 样式文件
└── README.md     # 说明文档
```

## 🚀 本地预览

直接用浏览器打开 `index.html` 文件即可预览。

## 🌐 在线访问

本站通过 **GitHub Pages** 托管，访问地址：

```
https://<your-username>.github.io/<repo-name>/
```

## ✏️ 自定义

- 修改 `index.html` 和 `about.html` 中的占位文字（姓名、介绍、联系方式等）
- 修改 `style.css` 中的 `:root` 变量可快速更换主题色
